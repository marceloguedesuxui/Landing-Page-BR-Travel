# ✈️ BR Travel
### Explore o Brasil com mais praticidade e inspiração

O **BR Travel** é um website/landing page criado para apresentar destinos turísticos brasileiros de forma visual, moderna e intuitiva.  
O projeto foi desenvolvido com foco em **UX/UI Design**, destacando experiências, roteiros e chamadas para ação voltadas ao turismo.

---

## 🚀 Tecnologias Utilizadas
- **HTML5**
- **CSS3 (Flexbox, Grid, Responsividade, Animações)**
- **JavaScript**
- **Google Fonts – Poppins**
- **Font Awesome (Ícones)**

---

## 🎯 Objetivos do Projeto
- Promover destinos turísticos do Brasil  
- Criar uma experiência visual imersiva  
- Facilitar a descoberta de viagens e roteiros  
- Demonstrar habilidades em **Landing Pages e UI turística**  
- Criar um projeto atrativo para portfólio

---

## ✨ Principais Recursos
- 🌍 **Hero section com imagens impactantes**  
- 🗺️ **Cards de destinos turísticos**  
- ⭐ **Seção de experiências e benefícios**  
- 🎠 **Carrossel de destinos ou avaliações**  
- 🖱️ **Efeitos de hover e animações suaves**  
- 📱 **Totalmente responsivo (mobile, tablet e desktop)**  
- 🎨 **UI moderna e inspiradora**
